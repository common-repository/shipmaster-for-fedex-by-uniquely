# Shipmaster for Fedex by Uniquely
This is a **FREE** FedEx Live Rate display plugin provided by [Uniquely Plugins](https://uniquelyplugins.com). 

Display live FedEx rates to your customers on cart page and checkout page, with easy integration.  Say goodbye to manually typing rates to zones or uploading via CSV.

This plugin supports a **single** *static* conversion rate (set in the settings menu), for when your store's currency is different to FedEx account currency.  However, multi-currency is not supported in this plugin.
Consider upgrading to the Pro version for features like multi-currency support, box packing, and automatic label creation.

== Features ==
* Automatic shipping costs calculation and display during cart and checkout.
* Based on order weight total and destination address.
* Choose between LIST and ACCOUNT rates.
* Fixed markup / markdown on shipping quotes.
* Include Or Exclude countries that can use this shipping method.
* Support single static currency conversion between store and FedEx.
* Uses FedEx REST API, it's brand new and it's faster and safer!

== Pro Features ==
* Supports dimensional weight.
* Automatic box packing with set box size.
* Separate package and combine quotes.
* Multi-currency support
* Automatic label creation (Automatically sends created label to your email address)

Consider upgrading to the PRO version to enjoy Pro Features.  (Currently in developemt.  Please check [Uniquely Plugins](https://uniquelyplugins.com) for more details.)

== Installation ==

This plugin can be easily installed like any other WordPress integration by following the steps below:

1. Download and unzip the latest zip file release.
2. Upload the entire plugin directory to your /wp-content/plugins/ path.
3. Activate the plugin using the Plugins menu in WordPress sidebar menu.

Optionally you can also try to upload the plugin zip file using Plugins -> Add New -> Upload Plugin option from the WordPress sidebar menu. Then go directly to point 3.

== Frequently Asked Questions == 
Q. Do I need a FedEx account?
A. Yes!  You'll also need to apply for a set of API credentials through [FedEX API portal](https://developers.fedex.com).
